# CP Progress Tracker

A centralized platform for tracking competitive programming progress across multiple coding platforms. The project aims to provide a unified dashboard where users can monitor their problem-solving journey, analyze strengths and weaknesses, revisit previously solved questions, and maintain an organized archive of coding solutions.

---

## Motivation

As someone preparing for coding interviews and competitive programming, I regularly solve problems on multiple platforms such as Codeforces, LeetCode, AlgoZenith, AtCoder, HackerRank, and others.

Each platform maintains its own statistics, ratings, contest history, and solved problem lists. This fragmentation makes it difficult to:

* Track overall coding progress.
* Analyze topic-wise strengths and weaknesses.
* Maintain a single archive of solved problems.
* Revisit previously solved questions efficiently.
* View learning progress across all platforms.

Although tools exist for individual platforms, I wanted to build a single system that aggregates data from multiple sources and provides meaningful insights into the entire coding journey.

---

## Objectives

The primary objective of this project is to build a centralized dashboard that enables users to:

* Track solved problems across multiple coding platforms.
* Organize problems using topics, difficulty, tags, and contests.
* Analyze coding progress through interactive statistics.
* Maintain a searchable archive of coding solutions.
* Identify weak topics requiring additional practice.
* Support structured revision of previously solved questions.
